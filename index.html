Libraire CSS/SASS - HtmlStyleConverter

Afin de limiter les mauvaises pratiques en incluant directement dans le code des balises style="", cette librairie permettra de générer directement votre style CSS à partir de vos classes.

Cette librairie n'est pas là pour remplacer le CSS mais d'optimiser le développement propre de vos applications.


Pour Exemple :

class="hsc padding-top_15px"

Donnera :

class {
    padding-top:15px;
}


Ou

class="hsc background-color_red width_25% padding-top_15px_xs"

Donnera :

class {
    background-color:red;
    width:25%;
    padding-top:15px; // seulement pour les tailles d'écrans < 768px
}


Dans ses exemples là, le premier argument (padding ou margin) stipule la condition, les seconds indiquent les arguments.

Cette librairie sera gérée avec une surcouche JavaScript, même si le choix de la technologie utilisée dépendra des solutions techniques.
